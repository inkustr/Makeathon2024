# DriveMate Backend

The bot is in prod and is ready to presentation.

Steps to launch:

1. import a project to vercel
2. [vercel] Change a version of node.js to 18.0
3. [vercel] Insert a gemini api key in .env variables
   3.1. If you need an api key contact us at oleg@mkpwnz.moe  
4. run `vercel --prod`

You also must run DriveMate Bot in own server. Link to a bot: https://github.com/oolxg/mktn-front
